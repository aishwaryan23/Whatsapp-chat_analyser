# Whatsapp-chat_analyser


A Streamlit-based Python application that analyzes your exported WhatsApp chat and provides interesting statistics, visualizations, and message insights.


## 🚀 Features

- 🔍 User-wise and overall chat analysis
- 🧮 Total messages, words, media shared, and links
- ⏰ Most active users and busiest days/times
- 🕗 Timeline of message frequency
- 💬 Word cloud of most frequent words
- 🛑 Stop word filtering
- 🔗 URL extraction
- 📈 Visualizations using Matplotlib and Seaborn


## 🛠️ Tech Stack

- Python 3.9+
- Streamlit
- Pandas
- Matplotlib
- Seaborn
- NLTK
- urlextract
- WordCloud

## Set up a virtual environment (optional but recommended)
- python -m venv venv
- source venv/bin/activate  # For Windows: venv\Scripts\activate

## Install dependencies
- pip install -r requirements.txt

## Run the app
 - streamlit run app.py
